## What This Is

A single-page static site deployed via GitHub Pages. It explains what Agentspace is, how to set it up, and provides the instruction file URL that users paste into their AI agent to get started.

## Development

Just open `index.html` in a browser. No build step, no dependencies.

## Deployment

Deployed automatically via GitHub Pages on push to `main`. Custom domain configured via `CNAME` file.
